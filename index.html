Numerical Methods For Chemical Engineers

Repository

This repository hosts the Python codes for the examples and algorithms discussed in Numerical Methods For Chemical Engineers. This book is a graduate level course in numerical methods for chemical engineers. This course has been taught by Prof. Vemuri Balakotaiah, who is also the main author of this book, at the University of Houston for many decades. I have had the good fortune of studying this course from him. The purpose of this book is to enable the reader (practitioner) to solve the differential and algebraic equations that arise in modelling of chemical processes by writing their own code in a language of their choice. This book is unique in its choice of material to equip the reader with the skillset necessary to understand, synthesize and analyze algorithms for solving novel research problems.

Why Python?

We choose Python as the language of choice because of its popularity and ease of understanding for a novice. Python has several libraries available which are wrappers to the widely used FORTRAN packages. In cases where Python's slower speed of execution becomes a hurdle, the just-in-time compilation, available through the Numba library comes in handy. If this also is not enough for the application then the critical code blocks can be executed in FORTRAN and C++ while the rest can be executed in Python.
